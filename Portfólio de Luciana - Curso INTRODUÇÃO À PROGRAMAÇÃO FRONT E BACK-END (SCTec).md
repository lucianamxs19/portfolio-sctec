# Portfólio de Luciana - Curso INTRODUÇÃO À PROGRAMAÇÃO FRONT E BACK-END (SCTec)

## 1. Visão Geral do Projeto

Este projeto é a **primeira landing page funcional** desenvolvida por Luciana, estudante do curso de **INTRODUÇÃO À PROGRAMAÇÃO FRONT E BACK-END** da SCTec. O objetivo principal é apresentar um portfólio pessoal, consolidando informações sobre sua formação, habilidades técnicas e comportamentais, e uma lista de projetos (reais ou fictícios) desenvolvidos ao longo do curso. A página serve como um instrumento inicial de apresentação profissional, projetada para ser facilmente atualizada e expandida.

## 2. Estrutura da Página

A landing page é organizada em três seções principais, conforme os requisitos do curso:

*   **Seção "Sobre"**: Apresenta o nome da estudante (Luciana), uma descrição pessoal focada em sua jornada na programação e o contexto do curso da SCTec. Inclui links diretos para o GitHub da Luciana.
*   **Seção "Habilidades"**: Detalha as competências adquiridas, divididas em Habilidades Técnicas (Hard Skills) como HTML5, CSS3, JavaScript e Git/GitHub, e Habilidades Comportamentais (Soft Skills) como resolução de problemas e trabalho em equipe.
*   **Seção "Projetos"**: Lista exemplos de projetos, desde os mais simples (como exercícios iniciais de HTML) até projetos mais complexos (como uma calculadora de IMC com JavaScript e a própria landing page), com descrições, tecnologias utilizadas e links para repositórios ou demonstrações.

## 3. Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais do desenvolvimento web, aprendidas no curso da SCTec:

*   **HTML5**: Utilizado para a estruturação semântica e acessível do conteúdo da página.
*   **CSS3**: Responsável pela estilização moderna, layout responsivo e adaptação a diferentes tamanhos de tela (desktop, tablet, mobile).
*   **JavaScript (ES6+)**: Implementado para adicionar interatividade à página, como rolagem suave entre as seções e destaque visual do menu de navegação conforme o usuário rola a página.

## 4. Como Executar o Projeto Localmente

Para visualizar e interagir com este portfólio em seu ambiente local, siga os passos abaixo:

1.  **Baixe o Projeto**: Faça o download do arquivo `portfolio_entrega.zip` e descompacte-o em uma pasta de sua preferência.

2.  **Navegue até a Pasta**: Abra o explorador de arquivos (ou terminal) e acesse o diretório onde os arquivos foram descompactados (ex: `portfolio/`).

3.  **Abra o `index.html`**: Localize o arquivo `index.html` dentro da pasta do projeto e abra-o diretamente em seu navegador web preferido (Google Chrome, Mozilla Firefox, Microsoft Edge, etc.). Não é necessário configurar um servidor web, pois este é um projeto estático.

    *   **Dica**: Você pode simplesmente arrastar o arquivo `index.html` para a janela do seu navegador.

## 5. Estrutura de Arquivos

O projeto está organizado da seguinte forma:

```
portfolio/
```
